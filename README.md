# Spot & Fix

Grammar practice with consequences for government writers. Built for the Washington State Department of Financial Institutions editing course.

## Modules

1. **One or Many?** — Subject-Verb Agreement
2. **The Reviewer's Red Pen** — Pronouns
3. **Not What You Meant** — Modifiers
4. **Same Words, Different Comma** — Commas

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Accessibility

- WCAG 2.2 AA color contrast throughout
- Full keyboard navigation with visible focus indicators
- Skip-to-content link
- ARIA roles, labels, and live regions
- Semantic HTML (buttons, not div-onClick)
- Screen reader–friendly button groups with role="group"

## Tech

- React 18 + Vite
- Josefin Sans + Source Serif 4
- No external component libraries
